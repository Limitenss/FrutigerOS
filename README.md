# Frutiger OS

A nostalgic web-based operating system inspired by the **Frutiger Aero** and **Windows 7** design language. Built with React and Vite, this project aims to capture the glassy, glossy, and optimistic aesthetic of the late 2000s.

## ✨ Features

- **Aero Glass UI**: Fully functional windowing system with transparency, reflections, and skeuomorphic details.
- **Authentic Windows 7 Elements**:
  - Interactive Taskbar with thumbnail previews.
  - Windows 7 style Start Menu with real-time search.
  - System Tray with flyout menus (Clock, Volume, Internet).
  - Aero Snap and Aero Shake window management.
  - Login & Startup sequences with authentic audio/visuals.
- **Built-in Applications**:
  - **Internet Explorer**: MSN-inspired news portal with historical events.
  - **Media Player**: Functional playback with a glossy 2009 UI.
  - **Minesweeper**: High-fidelity Aero implementation of the classic game.
  - **Photos**: Windows Photo Viewer clone.
  - **Pizzatron**: Classic web game integration.
  - **Personalization**: Change wallpapers, window colors, and sound schemes.
  - **Calculator & Notepad**: Pixel-perfect recreations.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/frugiter-os.git
   cd frugiter-os
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **React** - Component architecture
- **Vite** - Modern build tool
- **CSS3** - Custom glassy textures and animations
- **HTML5 Audio** - Authentic system soundscapes

## 🎨 Aesthetics

This project follows the **Frutiger Aero** design movement (c. 2004–2013), focusing on:
- Glossy "glass" surfaces
- Skeuomorphism
- Nature-tech fusion (water, grass, bubbles)
- Optimistic futurism
